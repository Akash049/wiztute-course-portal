Frontend Development with ReactJS

React is a library for building composable user interfaces. It encourages the creation of reusable UI components, which present data that changes over time . This course will teach you complete understanding of deploying a production ready webapp with ReactJS.

Duration: 20 days

Starting on: 29th March, 2020

Introduction
+

  • Introduction to ReactJS and setting up the development environment setup
  • Understanding the ES6 standards
  • Understanding the component structure and JSX
  • Understanding Git version control system with Github

React Configuration
+

  • Understanding the component lifecycle
  • Working with List and Conditionals
  • Working with React Developer Tools
  • Working with Higher Order Components

AJAX with React
+

  • Working with REST APIs and AJAX request in React
  • Understanding Router and Switch and working with navigation in React
  • Forms and Form Validation in React

Data Store and Accessibility
+

  • Introduction to Redux
  • Working with Redux Store, Reducers and Dispatching Actions
  • Updating the state and arrays immutably
  • Adding Authentication to React Webapp
  • Storing and updating the tokens dynamically and using the environment variables in React
  • Testing in React: Introduction to Jest and Enzyme

Deployment
+

  • Docker containers: Introduction
  • Generating a build in React and deployment with Docker container Image
  • Deployment on an AWS server. Working with Apache for routing in React
  • Working with React Hooks
